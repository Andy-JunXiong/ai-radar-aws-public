# AI Radar --- Knowledge Distillation Layer

## Goal

Transform high‑quality signals into reusable intelligence patterns.

Instead of only storing signals, AI Radar distills them into reusable
**analysis skills**.

------------------------------------------------------------------------

# Concept

Traditional pipeline:

Signal → Insight → Output

Distillation pipeline:

Signal → Insight → Pattern Extraction → Skill Library

------------------------------------------------------------------------

# Skill Layer Concept

A skill represents a reusable reasoning pattern derived from
high‑quality sources.

Examples:

-   evaluating new AI agent frameworks
-   analyzing AI infrastructure trends
-   assessing open‑source project momentum

------------------------------------------------------------------------

# Skill Schema

Example:

{ "skill_name": "Evaluate Agent Framework", "evaluation_questions": \[
"Is this a framework or wrapper?", "Does it enable reusable
infrastructure?", "Is there ecosystem adoption?" \] }

------------------------------------------------------------------------

# Distillation Sources

High quality signals come from:

-   top GitHub repositories
-   influential builders
-   technical essays
-   deep community discussions

AI Radar extracts reasoning patterns from these signals.

------------------------------------------------------------------------

# Distillation Pipeline

Signal ↓ Insight generation ↓ Pattern extraction ↓ Skill creation ↓
Skill library storage

------------------------------------------------------------------------

# Skill Library

Store skills in:

knowledge/skills/

Example files:

skills/ agent_framework_evaluation.md infra_trend_analysis.md
open_source_momentum.md

------------------------------------------------------------------------

# Integration With AI Radar

Signal ↓ Classifier ↓ Skill selection ↓ Skill‑guided reasoning

------------------------------------------------------------------------

# Benefits

-   reusable reasoning
-   consistent analysis
-   capture expert knowledge
